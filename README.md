# calcule-seu-imc


Este projeto é um site desenvolvido utilizando HTML e CSS, com foco na criação de um formulário moderno, responsivo e interativo.

O principal objetivo foi praticar conceitos fundamentais de desenvolvimento web, como estruturação de páginas, estilização com CSS e melhorias na experiência do usuário. O formulário conta com elementos visuais bem organizados, além de interações como mudança de estado de botões quando os campos não estão preenchidos corretamente.

Durante o desenvolvimento, foram aplicadas técnicas como:

Uso de Flexbox para alinhamento de elementos
Estilização personalizada de inputs e botões
Feedback visual para o usuário (ex: botão desativado quando necessário)
Organização de layout de forma limpa e responsiva

Este projeto é ideal para fins de estudo e demonstração de habilidades básicas em front-end.
